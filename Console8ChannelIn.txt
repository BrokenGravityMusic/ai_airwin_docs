Console8ChannelIn


Converts mono or stereo input to 8-channel submix, with added analog simulation.


Key Controls and Settings:

* `Input Type:` Select the input type (mono or stereo).
    + Range: N/A
    + Type: Dropdown menu
* `Simulator Type:` Choose from different analog simulation types.
    + Range: N/A
    + Type: Dropdown menu
* `Mix Control:` Blend the original input signal with the simulated output.
    + Range: 0% to 100%
    + Type: Fader control
* `Gain:` Adjust the overall gain of the simulated output.
    + Range: Adjustable 
    + Type: Gain control

Best Use Cases:
Ideal for converting mono or stereo sources to a 8-channel submix, adding a touch of analog warmth. Suitable for mixing and mastering applications.

Tips on Optimal Settings:
To achieve the best results:
* Experiment with different simulator types to find the desired tone.
* Adjust the mix control to balance the original input signal with the simulated output.
* Use the gain control to optimize the overall level of the simulated output.
