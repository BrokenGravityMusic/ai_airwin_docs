Console8SubIn

Converts an input bus into a submix topology with analog simulation

Key Controls and Settings:

* Channel/Submix/Buss Selection: Choose the type of topology (e.g. channel, submix, buss)
	+ Range: N/A
	+ Type: Dropdown menu
* Analog Simulation Type: Select from various analog simulation models
	+ Range: N/A
	+ Type: Dropdown menu
* Gain: Adjust the overall gain of the processed audio
	+ Range: N/A
	+ Type: Knob or slider

Caveats:
None noted.

Best Use Cases:

* Create a submix topology for mixing and mastering
* Add analog warmth to an input bus
* Mixing and mastering applications where a channel/submix/buss topology is desired

Tips on Optimal Settings:
Experiment with different submix topologies and analog simulation types to find the best configuration for your sound. Adjust gain to taste, taking care not to overdrive the signal.
