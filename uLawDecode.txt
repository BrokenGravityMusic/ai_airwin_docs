uLawDecode

A console-like encode/decode pair that simulates the extreme compression of uLaw audio encoding.

Key Controls and Settings:

* `Mode:` Select between "encode" or "decode" modes.
    + Type: Mode
* `Input/Output:` Choose whether to input or output a signal.
    + Range: Input/Output selection
    + Type: Switch
* `Bit Depth:` Set the bit depth for encoding (default: 8 bits).
    + Range: 1-16 bits
    + Type: Slider
* `Sampling Rate:` Set the sampling rate (default: 8000 Hz).
    + Range: 8 kHz, 11.025 kHz, 22.05 kHz
    + Type: Dropdown

Best Use Cases:

* Simulate extreme compression of audio signals.
* Test the limits of audio encoding/decoding algorithms.
* Decoding compressed u-Law audio files or streams.
* Converting between different sampling rates using the paired encoder/decoder.
