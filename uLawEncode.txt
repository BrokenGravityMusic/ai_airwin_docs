uLawEncode
`Encoding audio using uLaw compression algorithm. Applies distortion to quiet sounds and squishes dynamic range.`

Key Controls and Settings:

* Gain control: Adjusts the overall level of the encoded signal.
    + Default value: 1.0
* Wet/Dry control: Controls the amount of encoded signal blended with the original audio.
    + Default value: 1.0

Best Use Cases:

* As a console-like encode/decode pair with bitcrushers
* As a distortion plugin to create unusual sounds
* As a building block for creative signal processing chains
